Printer State Tool
PrinterStateTool

Provides all installed and available printers information. The information includes printer queue info and printer properties info for every printer.
PrinterState - Main library. Provides all functionality for getting printers information.
AllPrintersStates - Console application. Provides all printers queue info.
PrinterStateTool - Winforms application. Comfortable tool for getting information about every installed printer.

Asterisk Connector Tool
AsteriskConnector

Asterisk PBX connector tool. Uses AsterNET library.

Crm Test Email Sender Tool
CrmTestEmailSender

Sends email message from one CRM user to another users. Works for Microsoft Dynamics CRM (2011-2016) and Dynamics 365.
Sends email message with Subject "Test subject [current_date]" and Description "Test message [current date]" to provided users and shows sent email Id if succeed.


Crm Solutions Info Helper Tool
CrmSolutionsInfoHelper

Shows information about active custom solutions imported in CRM. Works for Microsoft Dynamics CRM (2011-2016) and Dynamics 365.
Available information about solutions: name, version, is managed or not, date installed, date modified.


Crm Max Records For Export To Excel Editor Tool
CrmMaxRecordsForExportToExcelEditor

Edits the max count of records in CRM page for exporting to Excel. Works for Microsoft Dynamics CRM (2011-2016) and Dynamics 365.
By default, you can export a maximum of 10,000 records from Microsoft Dynamics CRM to an Excel worksheet by using the "Static worksheet with records from all pages in the current view" feature.
If you export more than 10,000 records, only the first 10,000 records are exported. You do not receive a warning message that lets you know that only the first 10,000 records were exported.
