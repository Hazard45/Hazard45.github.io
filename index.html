Printer State Tool

Provides all installed and available printers information. The information includes printer queue info and printer properties info for every printer.
● PrinterState - Class library. Provides all functionality for getting printers information
● AllPrintersStates - Console application. Provides all printers queue info
● PrinterStateTool - Winforms application. Comfortable tool for getting information about every installed printer

Armenian Text To Speech Library

As we know, there is no Text-to-Speech functionality for Armenian language in Windows.
The application uses System.Speech.Synthesis library, which supports Text-To-Speech functionality for English language and provides it for Armenian.
● ArmenianTextToSpeech - Class library. Provides Text-to-Speech functionality for Armenian language. It allows to convert armenian words to speech with different speed
● Visualizer - Winforms application. Comfortable tool for testing the library

Asterisk Connector Tool

Asterisk PBX connector tool. Uses AsterNET library.
Allows to:
● Set manager connection with Asterisk
● Originate and decline calls
● Set and get DND state
● Execute CLI commands
● Catch and view all events

CRM Test Email Sender Tool

Sends email message from one CRM user to another users. Works for Microsoft Dynamics CRM (2011-2016) and Dynamics 365.
● You should provide CRM users IDs
● Email subject is fixed: "Test subject [current_date]"
● Email description is fixed: "Test message [current date]"
● Returns sent email ID if succeed

CRM Solutions Info Helper Tool

Shows information about active custom solutions imported in CRM. Works for Microsoft Dynamics CRM (2011-2016) and Dynamics 365.
Provides:
● Solution name
● Solution version
● Is solution managed or not
● Solution installation date
● Solution modification date

CRM Max Records For Export To Excel Editor Tool

Edits the max count of records in CRM page for exporting to Excel. Works for Microsoft Dynamics CRM (2011-2016) and Dynamics 365. By default, you can export a maximum of 10,000 records from Microsoft Dynamics CRM to an Excel worksheet by using the "Static worksheet with records from all pages in the current view" feature. If you export more than 10,000 records, only the first 10,000 records are exported. You do not receive a warning message that lets you know that only the first 10,000 records were exported.
● You should provide new maximum records count for Excel export
